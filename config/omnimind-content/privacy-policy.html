OmniMind 隐私政策

最后更新：2026 年 5 月 | Last Updated: May 2026

引言

OmniMind — 万象衍智（以下简称"我们" "平台"）深知个人信息对您的重要性，并将按照法律法规的要求，采取相应的安全保护措施，尽力保护您的个人信息安全可控。本隐私政策旨在向您说明我们如何收集、使用、存储、保护和共享您的个人信息。

1. 我们收集的信息

1.1 账户信息

注册账户时，我们收集您的用户名、邮箱地址和（可选的）显示名称。邮箱地址用于账户验证、密码重置和安全通知。

1.2 使用数据

我们收集 API 调用元数据：请求时间戳、模型名称、Token 消耗量、响应状态码。用于计费、用量统计和服务优化。
重要：我们不收集、不存储、不留存您通过 API 发送的提示词 (Prompt) 或模型返回的生成内容。

1.3 支付信息

支付由第三方支付网关（支付宝、微信支付、EPUSDT）处理。我们不直接收集或存储您的银行卡号、密码等敏感支付信息。

1.4 技术数据

我们记录 IP 地址用于速率限制、安全防护和地理负载均衡。日志按最小必要原则保留，定期自动清除。

2. 信息使用方式

我们使用收集的信息严格限于以下目的：

  • 提供、维护和改进服务
  • 处理交易和发送账单通知
  • 发送服务相关通知（安全、维护、政策更新）
  • 检测、预防和解决技术问题及安全事件
  • 遵守法律义务

⚠️ 我们不会：出售您的个人信息、将 API 数据用于训练模型、或在未经您明确同意的情况下向第三方共享您的数据（法律要求除外）。

3. 数据存储与安全

3.1 存储位置：用户数据存储于亚太地区（主要节点位于新加坡和中国香港）的安全服务器上。我们遵守 GDPR、PIPL 等国际数据保护法规。

3.2 加密措施：传输中数据使用 TLS 1.3 加密；API Key 使用 SHA-256 哈希存储；数据库启用静态加密。

3.3 保留期限：账户信息在账户存续期间保留；API 调用日志保留 30 天滚动清除；支付记录依法保留 5 年。

3.4 访问控制：仅经授权的运维人员可通过严格审计的通道访问生产数据，所有操作完整记录。

4. 信息共享与披露

4.1 除以下情形外，我们不会与第三方共享您的个人信息：

· 服务提供商：支付网关、邮件服务、CDN 等 — 仅提供为完成服务所必需的最少数据
· 法律要求：依法向司法机关、监管部门披露
· 业务转移：如发生合并、收购或资产出售，用户信息可能作为资产的一部分转移
· 经您同意：在获得您明确授权后

4.2 我们在共享前与第三方签署严格的数据保护协议，确保其安全标准不低于本政策要求。

5. 您的权利

根据适用法律，您享有以下权利：

  • 访问权：查阅我们持有您的个人信息
  • 更正权：要求更正不准确或不完整的个人信息
  • 删除权（被遗忘权）：在特定条件下要求删除您的个人信息
  • 数据可携带权：获取结构化、通用、机器可读格式的个人数据副本
  • 限制处理权：在特定条件下限制我们对您数据的处理
  • 反对权：反对基于直接营销目的的数据处理

如需行使上述权利，请发送邮件至 privacy@ulnit.com，我们将在 30 日内回复。

6. Cookie 与追踪技术

我们使用必要的 Cookie 来维持登录会话和基础平台功能。我们不使用第三方追踪 Cookie、广告定位 Cookie 或跨站点追踪。

我们使用自托管的 Umami Analytics 进行匿名页面访问统计（不涉及个人信息、不设置 Cookie、数据归我们完全控制），用于了解功能使用情况和优化用户体验。

7. 未成年人保护

本服务不针对 13 周岁以下的儿童。我们不会在知情的情况下收集 13 周岁以下儿童的个人信息。如发现误收集，将立即删除。如您是监护人并认为我们持有您监护对象的信息，请立即联系我们。

8. 国际数据传输

您的信息可能在您所在国家/地区以外的服务器上处理和存储。我们采取标准合同条款 (SCC) 等适当保护措施，确保数据在传输过程中获得与所在司法管辖区同等水平的保护。使用本服务，即表示您同意此等传输。

9. 隐私政策更新

我们可能会不时更新本隐私政策。重大变更将通过平台公告、邮件或登录弹窗等多种方式通知您。本政策顶部的"最后更新"日期将同步修改。继续使用服务即表示您接受更新后的隐私政策。
