使用 OpenAI 兼容接口

API 接入示例

# 使用 OpenAI 兼容接口
curl https://ai.ulnit.com/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-4o","messages":[{"role":"user","content":"你好"}]}'

# Python 示例
import openai
client = openai.OpenAI(api_key="sk-your-api-key", base_url="https://ai.ulnit.com/v1")
response = client.chat.completions.create(model="gpt-4o", messages=[{"role":"user","content":"你好"}])
